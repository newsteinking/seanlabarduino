

### Welcome to SEANLABArduino

this is forked from BlocklyDuino( https://github.com/BlocklyDuino/BlocklyDuino)


이 프로젝트의 목적은 BlocklyDuino를 seanlabcoding에 활용하고자 하는 목적입니다.

### Install & Configuration
1. install python 2.x or 3.x

2. clone
  (prework: install git)
 

   >git clone https://github.com/newsteinking/seanlabarduino

   >cd seanlabarduino
  
   >python arduino_web_server.py

3. web browser
  http://127.0.0.1:8080
